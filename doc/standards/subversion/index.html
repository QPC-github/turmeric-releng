Subversion Configuration

In order to benefit yourself, and all of the other developers within the Turmeric project, it is strongly advised that you configure your own subversion client with at least some basic [auto-props].

Windows: TortoiseSVN Configuration

To configure your TortoiseSVN config file, open up the "Settings" menu choice in the standard right click menu on your windows explorer folders.

Next, use the button "Edit" for the "Subversino configuration file" to open up your notepad editor to the configuration file that TortoiseSVN uses.

Ensure you have enable-auto-props = yes set in your configuration file.

Linux

In linux this configuration is easy.

  1. Edit your $HOME/.subversion/config file
  2. Ensure you have enable-auto-props = yes set
  3. Add the missing [auto-props] entries from the example above
